OTAA Dashboard

A web app that displays information about the vulnerability and threats associate with devices. I used React and MUI framework and Google progressbar and other plugins to create the layout and design of the Dashboard. In this project I learn how to divide the whole code into functional components and make cleaner and presentable.

Tic-Tac-Toe_Game

Tic-Tac-Toe is a simple and fun game that you can play with your friends. I created this project using react,useState hooks and css. A responsive and interactive user interface that adapts to different screen sizes and devices. I learned how to use React hooks to create functional components and manage state without using classes or lifecycle methods. I learned how to use CSS grid and media queries to create a responsive and flexible layout for the game board.

WorkOut-todo-List

Workout Todo List is a web app that helps you plan and track your workouts. You can create, edit, and delete your workout tasks. I built this app using React and some plugins, such as React Hooks, React Router. I used React Hooks to manage the state and the effects of the app. I used React Router to create different routes and pages for the app. I stored the data in the local storage of the browser.
